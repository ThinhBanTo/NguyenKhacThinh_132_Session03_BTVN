# 1+3: Xác định thực thể, thuộc tính, khóa chính, khóa ngoại và kiểu liên kết
* Các thành phần:
  
  Thực thể|Thuộc tính|Khóa chính|Khóa phụ
  --|--|--|--
  Khách hàng|Mã khách hàng,họ tên, email, sđt, địa chỉ|Mã khách hàng|X
  Đơn hàng|Mã đơn,ngày tạo, tổng tiền, mã khách hàng|Mã đơn|Mã khách hàng
  Sản phẩm|Mã sản phẩm, tên, giá, loại sản phẩm|Mã sản phẩm|X
  Chi tiết đơn hàng|Số lượng, đơn giá, mã sản phẩm, mã đơn hàng| Mã sản phẩm+mã đơn hàng|Mã sản phẩm, mã đơn hàng

* Kiểu liên kết:
  - Quan hệ N-M:
    + Đơn hàng - Có - Sản phẩm
  - Quan hệ 1-N:
    + Đơn hàng - Có - Chi tiết đơn hàng
    + Sản phẩm - Thuộc - Chi tiết đơn hàng
    + Khách hàng - Đặt - Đơn hàng
